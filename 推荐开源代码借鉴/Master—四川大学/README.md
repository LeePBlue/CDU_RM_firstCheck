# Master 核心代码程序

## 项目说明

**Master 核心代码程序**是RoboMaster超级对抗赛的主运行流程，记录了程序从比赛开始到结束的核心代码。

本程序主要作用是调度其他模块，进行线程管理，数据和消息同步，实现多线程进程的同步和异步处理。
